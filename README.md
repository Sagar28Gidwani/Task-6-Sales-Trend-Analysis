### Dataset:
`orders` table with:
- order_id
- order_date
- amount
- product_id

### SQL Concepts Used:
- `EXTRACT()` for month & year
- `SUM()` for revenue
- `COUNT(DISTINCT order_id)` for order volume
- `GROUP BY` and `ORDER BY` for trend analysis
- `LIMIT` for top 3 results

### Deliverables:
- SQL script (`task6_sales_trend.sql`)
- Results table (CSV/Markdown)
- README.md (this file)
